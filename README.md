# 💕 Anniversary Game Hub (1726web)

一款专为情侣纪念日设计的、移动端优先的互动 Web 应用。集成了 10 款创意小游戏、情感互动模块、惊喜礼盒以及成就系统，旨在记录和庆祝彼此的每一个闪光时刻。

---

## ✨ 核心功能

### 🎮 游戏中心 (10 个关卡)
1.  **爱know (Quiz)**：关于彼此的小知识测试，答对触发心形礼花。
2.  **回忆重组 (Puzzle)**：基于 `@dnd-kit` 的物理切割拼图，支持上传年度合照。
3.  **雷霆战机 (Fighter)**：重力感应/键盘双控射击，包含永久 Buff 商店。
4.  **数独挑战 (Sudoku)**：中等难度逻辑游戏，移动端友好型键盘。
5.  **2048之恋 (2048)**：将数字转化为情感阶段，合成“永恒”即为胜利。
6.  **记忆方块 (Memory)**：翻转配对浪漫 Emoji，测试你们的灵犀程度。
7.  **节奏大师 (Rhythm)**：跟随节奏点击音符，高能 Combo 体验。
8.  **飞翔的心 (Flappy)**：操控爱心穿过障碍，获取高分。
9.  **接住这份爱 (Collector)**：敏捷类游戏，接住落下的心，避开炸弹。
10. **天生一对 (Sync Quiz)**：双人轮流操作，计算最终的默契匹配度。

### 💌 情感模块
*   **📖 我们的故事**：交互式纵向时光轴，记录从初见到现在的每一个重要里程碑。
*   **📌 留言板**：可随时发布的电子便利贴，支持多色分类，持久化保存。
*   **🖼️ 永久相册**：支持图片上传、预览与 Caption 描述，留住灿烂瞬间。
*   **✉️ 神秘情书**：定时锁定的信件，仅在每年 1 月 30 日（纪念日当天）自动解锁。

### 🏆 游戏化机制
*   **🎁 惊喜礼盒**：使用游戏中赚取的金币，兑换“洗碗券”、“大餐预约”等实体奖励。
*   **🏅 成就系统**：追踪全站进度，解锁“小富翁”、“通关专家”等勋章。
*   **🔥 每日挑战**：每天随机推荐一款游戏，完成后获得额外 +100 💰 奖励。

## 🛠️ 技术栈

*   **框架**: Next.js 15 (App Router)
*   **语言**: TypeScript (Strict Mode)
*   **动画**: Framer Motion & Animate.css
*   **样式**: Tailwind CSS
*   **音效**: Web Audio API (自定义 SoundProvider)
*   **状态管理**: LocalStorage 自动同步 (lib/game.ts)
*   **物理/工具**: `@dnd-kit` (拖拽), `canvas-confetti` (礼花)

## 🚀 快速开始

### 1. 克隆与安装
```bash
git clone <your-repo-url>
cd 1726web
npm install
```

### 2. 开发环境
```bash
npm run dev
```
访问 [http://localhost:3000](http://localhost:3000) 即可预览。默认访问口令为：`love`。

### 3. 构建与部署
```bash
npm run build
npm start
```

## 🔑 开发者后门 (Debug)
在“游戏关卡”选择页：
*   **解锁当前关卡**：长按对应的关卡卡片 10 秒。
*   **一键解锁全关**：长按页面顶部的标题“🎮 游戏关卡” 5 秒。

---

## 📁 目录结构
*   `/app`: 核心页面路由（games, gifts, story, gallery...）
*   `/components`: 通用 UI 组件（如 Modal.tsx）
*   `/hooks`: 逻辑钩子（useSound, useModalConfig）
*   `/lib`: 全局状态管理、进度持久化逻辑及配置 (game.ts)
*   `/public`: 静态资源（音效、占位图）
